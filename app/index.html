3+2 郵遞區號查詢


你查詢的郵遞區號是

快速

只要輸入完整的地址資料, 就可以查出3+2碼的郵遞區號 (地址中間不要有空白) 本查詢是根據 中華郵政 的資料為基礎, (支援六都地址) 輸入台北縣的地址, 查詢結果會自動轉成新北市 (台中縣,台南縣,高雄縣 也一樣) 2015-01-05 也支援 桃園縣升格為桃園市後 的地址郵遞區號 巷弄號, 請用半型數字 請注意, 查詢出的結果有時會有錯誤, 輸入的地址不完整, 查詢出的結果就會不正確 如果查詢結果空白, 代表查不出你輸入的地址的郵遞區號, 可以到 中華郵政 去查詢 如何在自己的網頁裡使用 3+2郵遞區號 的 api 呢??請按此 使用 api 的朋友, 請在每次查詢間都留一段緩衝時間, 且勿連續長時間的查詢, 感謝 依縣市來點選查詢 請按這裡 ,要查詢 3碼 郵遞區號 請按這裡
